Requirements

  • Python 3.8+
  • Dependencies: PyQt5, numpy, scipy, matplotlib, Dans_Diffraction (see requirements.txt)

Install

python -m venv .venv
source .venv/bin/activate  # or .venv\Scripts\activate on Windows
pip install -r requirements.txt

Run

python -m advisor
# or
python main.py

Minimal workflow

  1. Launch the app.
  2. Enter lattice constants/angles and beam energy, or drop a CIF file.
  3. Click Initialize.
  4. Switch between feature tabs (Brillouin / Structure Factor) to calculate and visualize.
