Welcome

Read-friendly documentation for the PyQt5-based Ad-VISOR. Use these pages to install, run, and extend the app without diving deep into code.

Tip: These docs are light on theory and heavy on steps. For developer notes, see Extending the App.

  • Project Notes — design choices, roadmap ideas, and media placeholders.

What this app does

The toolbox streamlines X-ray scattering/diffraction preparation: initialize lattice parameters (or drop a CIF), compute and visualize scattering geometry, convert angles ↔ HKL, and inspect structure factors across planes.
